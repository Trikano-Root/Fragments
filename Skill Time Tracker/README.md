# Skill Time Tracker (Tracker.io)

A lightweight utility for the observation and quantification of skill acquisition. 

### Implementation
This tool was built to eliminate the friction inherent in manual time-logging. It functions as a local, browser-based ledger designed for speed and minimal cognitive overhead.

### Usage
1. Open `index.html` in any modern web browser.
2. Input the target skill and initiate the timer.
3. Upon completion, the utility logs the session, providing a graphical view of time investment.
4. All sessions are stored via browser `localStorage`.
   * **Note:** As data is local to this browser, please use the **Export/Import** buttons provided in the UI to backup your logs or migrate them if you clear your browser cache.

> "What gets measured gets improved."
